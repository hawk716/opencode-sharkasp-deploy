# OpenCode Deployment for SharkASP

هذا المستودع مخصص لنشر **OpenCode** على منصة SharkASP.NET أو أي بيئة مشابهة تدعم Node.js.

## التعديلات الجديدة

تم تحديث المشروع ليدعم:
- إضافة `$HOME/.opencode/bin` إلى الـ `PATH` تلقائياً.
- أمر التشغيل المحدث: `opencode web --hostname 0.0.0.0 --port 3000 --password o`.
- تبسيط عملية التثبيت لتتم عبر `curl` أو `npm` تلقائياً.

## البدء في العمل

لتشغيل المشروع، كل ما عليك فعله هو تشغيل:

```bash
npm start
```

سيقوم السكربت تلقائياً بـ:
1. محاولة تثبيت OpenCode عبر `curl -fsSL https://opencode.ai/install | bash`.
2. إذا فشل، سيحاول التثبيت عبر `npm install -g opencode-ai`.
3. تشغيل خادم الويب الخاص بـ OpenCode مع الإعدادات المطلوبة.

## الإعدادات

- **المنفذ (Port):** يستخدم المتغير `PORT` من البيئة أو الافتراضي `3000`.
- **كلمة المرور:** الافتراضية هي `o`.
- **المضيف (Hostname):** `0.0.0.0` للسماح بالوصول الخارجي.
